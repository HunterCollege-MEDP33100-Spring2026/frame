# **Assignment: Frame**

Complete the webpage template that fetches data from a public API and displays it in the “frame” element.

### **Requirements**

In the `script.js`:

1. Pick a free, public API (e.g. NASA, Dog API, PokéAPI, etc.)
    - You can pick an API from this list: https://free-apis.github.io/#/browse
2. In `getData()`, use `fetch()` to fetch data from the API
3. Insert 10+ `li` elements into the `.frame` element
